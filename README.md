# 2d-Robot-Localization
The program itself runs a simulation and ouputs you the results.

How to use:
Run the program, it will generate a .gif of the simulation. 
Any parameters to be tinkered with are at the very beginging of the program's .py file. 

Things to note:
Make sure you have the "Maze#.png" in the same folder as the actual program. 
Time updates will be printed as the program runs.
